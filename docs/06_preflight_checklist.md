# Preflight 체크리스트 (v1.0)

> 목표: 인쇄 사고(폰트 치환/박스 오류/깨짐) 방지

## 1) 자동 검사 항목(MVP 포함)
- [ ] PDF 내 폰트가 **지정 폰트만** 사용되었는가?
- [ ] 폰트가 **임베딩(embedded/subset)** 되었는가?
- [ ] TrimBox/BleedBox가 규격대로 존재하는가?
- [ ] 페이지 크기/bleed 값이 템플릿 규격과 일치하는가?
- [ ] 다국어 샘플(베트남어 성조/악센트/ñ) 회귀 테스트가 통과하는가?

## 2) PDF/X 관련(우대/협의)
- PDF/X-1a 또는 PDF/X-4 완전 준수는
  - 출력 의도 ICC, 트래핑, 오버프린트, 투명도 처리 등
  - 인쇄소 및 RIP 설정과 맞물려 검수 범위가 커질 수 있음
- MVP에서는 **PDF/X 대응 가능한 구조 + preflight 리포트** 제공을 기본으로 하고,
  완전 준수 보증은 2차 확장으로 권장합니다.

## 3) 레포 제공 스크립트
- `scripts/preflight-check.js` (샘플)
  - PDF 파일을 대상으로 폰트/박스/메타 검사(기본 검사)
  - 실제 인쇄소 프리플라이트 도구(예: Acrobat Preflight, Callas 등) 연동은 2차 확장 포인트
