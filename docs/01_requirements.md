# VDP MVP — 요구사항/범위 정의 (v1.0)
작성일: 2026-02-19

## 1) 목표
사용자가 선택한 **4개 언어(메인 1 + 서브 3)** 조합과 개인화 데이터(이름/날짜)를 서버에서 실시간 치환하여,
상업 인쇄용 PDF(8.5x11 inch, bleed 포함, 폰트 임베딩)를 생성/저장/재생성한다.

## 2) 포함 범위(MVP 고정)
- 다국어 자동 조판 로직(8개 국어 DB 기반)
- 4개 언어 조합에 따른 레이아웃 구성
- 개인화 치환: **Cover/Opening/Closing 3페이지 한정**
- 관리자(Admin): 원고/템플릿/폰트프리셋 CRUD, 주문/잡/결과 관리
- PDF 생성 엔진(폰트 임베딩, bleed/box 설정 포함)
- 서버 배포 가이드(Docker)

## 3) 제외 범위(2차 확장/별도 견적)
- AI 이미지 생성/얼굴 합성/영상/음성 변환
- 글로벌 결제/정산/모바일 앱
- 본문(스토리) 페이지 개인화 확대
- 고급 하이픈(hyphenation)/금칙처리 고도화/RTL shaping(아랍어 등)
- PDF/X 완전 준수(ICC, 트래핑, 오버프린트 포함) 보증

> 변경관리: 제외 범위는 Change Request로 처리하며 일정/비용 재산정 후 2차 확장으로 진행한다.

## 4) 수용 기준(검수)
- UTF-8 다국어 특수문자 출력 오류 0% (깨짐/누락/대체 폰트 0건)
- 지정 폰트 Full Embedding(라이선스 확인된 ttf/otf)
- bleed/trim box가 명확하고 프리프레스에서 확인 가능
- 줄바꿈(Line-break)만 수행(Auto-shrink 미적용)
- 동일 입력(버전 동일) 재생성 시 결과가 재현 가능
